# スポーツ認知反応トレーニング — デザイン設計書

## コンセプト
スポーツ競技力向上を目的とした端末内完結型認知反応トレーニングアプリ。
派手な演出よりも「継続しやすさ」「見やすさ」「判定精度」を最優先とする。

## 画面一覧

### 1. ホーム画面 (HomeScreen)
- アプリ名・ロゴ
- 「トレーニング開始」ボタン（大・グリーン）
- 「記録」ボタン
- 「設定」ボタン
- バナー広告（下部）

### 2. 初回オンボーディング (OnboardingScreen)
- 利用方法説明（スワイプ式3ページ）
- カメラ権限リクエスト
- 「始める」ボタン

### 3. 難易度選択画面 (DifficultyScreen)
- 初級（3秒）/ 中級（2秒）/ 上級（1秒）/ エリート（0.7秒）
- 各難易度カード（タップで選択）
- 「開始」ボタン

### 4. カメラキャリブレーション画面 (CalibrationScreen)
- カメラプレビュー（全画面）
- 「全身が映る位置へ移動してください」ガイドオーバーレイ
- 骨格検出フィードバック（緑枠）
- 認識成功 → 自動遷移

### 5. トレーニング画面 (TrainingScreen)
- カメラプレビュー（背景・半透明）
- 認知課題表示（大文字・中央）
- 身体動作課題表示
- タイマーバー（上部）
- 判定フィードバック（成功/失敗/タイムオーバー）
- コンボカウンター
- スコア（右上）

### 6. 結果画面 (ResultScreen)
- ランク表示（S/A/B/C/D）（大・中央）
- 総得点
- 成功率
- 平均反応時間
- 最大コンボ
- 「もう一度」ボタン
- 「結果をシェア」ボタン
- 「ホームへ」ボタン

### 7. 記録画面 (RecordsScreen)
- ベストスコア・総プレイ回数・累計時間（上部サマリー）
- 過去履歴一覧（FlatList）
- バナー広告（下部）

### 8. 設定画面 (SettingsScreen)
- 音声ON/OFF
- ダークモード切替
- バナー広告（下部）

---

## 主要ユーザーフロー

### 初回起動フロー
オンボーディング → カメラ権限 → ホーム

### トレーニングフロー
ホーム → 難易度選択 → キャリブレーション → トレーニング → インタースティシャル広告（2回に1回）→ 結果 → ホーム

### 記録確認フロー
ホーム → 記録画面 → 履歴詳細

## タイポグラフィ
- 認知課題テキスト: 72sp Bold（視認性最優先）
- ボタンラベル: 18sp SemiBold
- スコア数値: 48sp Bold
- 説明テキスト: 16sp Regular

## レイアウト原則
- ボタン最小タップ領域: 56×56pt（iPad: 64×64pt）
- 余白: 16pt（iPhone）/ 24pt（iPad）
- 縦横両対応: SafeAreaView + useWindowDimensions で動的調整
